# KisaanBazaar

Monorepo for KisaanBazaar marketplace platform.

## Workspace

- apps/api: Express + TypeScript backend
- apps/web: React + TypeScript frontend
- packages/shared: Shared contracts and types

## Quick start

1. Install dependencies:
   npm install
2. Run API:
   npm run dev:api
3. Run web:
   npm run dev:web
